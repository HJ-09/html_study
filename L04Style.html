CSS 스타일

브라우저는 html을 출력할 때 CSS를 기반으로 적용합니다.

스타일의 종류

  • color : text 색 자식요소는 부모요소의 스타일을 그대로 적용(cascading)
  • background-color : 배경 색 #000000 (00 red 00 green 00 blue) ⇒ 16진수 표기법
  • font-size : 폰트 크기, -weight : 폰트 굵기
  • border : 테두리, contents의 경계
  • text-align : 텍스트 정렬 [left, center, right]
  • margin : 외부(바깥)여백 / padding : 내부(안쪽)여백
