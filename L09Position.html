position으로 요소의 위치를 조정하자

position:relative로 이동하기

→ 일반적(static)으로 배치된 위치를 기준으로 이동함

position:absolute(절대적인, 독립적인)

→ 독립적인 존재이기 때문에 브라우저 기준임

absolute box
top: 0; left: 0
absolute box
bottom: 0; right: 0

absolute 위치의 기준을 부모의 relative로 변경

→ 부모의 position이 relative이기 때문에 자식의 position 기준이 부모가 0,0이 됨

absolute

fixed:view port의 특정 위치에 고정

absolute와 fixed는 독립적으로 존재하기 때문에 너비의 기준인 부모요소가 없어서 강제로 지정해줘야함.
