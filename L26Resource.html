리소스 참조를 배워보자

http://localhost:63342/HTML_study/L26Resource.html

  • http ⇒ 통신규약
  • localhost ⇒ 내 컴퓨터 내부 127.0.0.1
  • 63342 ⇒ os가 부여한 고유번호 (=port)
  • HTML_study/L26Resource.html ⇒ 서버 내의 리소스 주소(path)
  • http://localhost:63342/HTML_study/L26Resource.html ⇒ url

상대 경로

  • /path ⇒ 루트경로. 리소스의 맨 위 상단

다른 문서(html)를 포함하는 iframe

/ : 루트 상대경로

./ ../ : 리소스 위치 기준 상대경로

절대경로 ⇒ 통신규약 or 드라이버

절대경로로 다른 사이트 참조

CORP: Cross-Origin Resource Policy ⇒ SOP(Same Origin Policy): 같은 서버의 리소스만 참조가능.
